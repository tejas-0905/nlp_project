# 🎬 Movie Review Sentiment Analysis using NLP

This project performs **Sentiment Analysis on Movie Reviews** using **Natural Language Processing (NLP)** techniques.  
The model classifies reviews as **Positive** or **Negative** based on the text content.

The system preprocesses text, converts it into numerical features using **TF-IDF**, and applies a **Machine Learning model** to predict sentiment.

---

## 🚀 Features

- Movie review sentiment classification  
- Text preprocessing (cleaning, tokenization)  
- Stopword removal  
- TF-IDF vectorization  
- Machine learning classification  
- Real-time sentiment prediction  
- Accuracy evaluation  
- Easy to understand notebook implementation  

---

## 🛠️ Tech Stack

- Python  
- NLP (Natural Language Processing)  
- Scikit-learn  
- Pandas  
- NumPy  
- TF-IDF Vectorizer  
- Logistic Regression / Naive Bayes  
- Jupyter Notebook
